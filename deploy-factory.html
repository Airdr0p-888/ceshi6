📖 什么是 VanityFactory？

VanityFactory 是一个基于 CREATE2 的工厂合约。部署代币时，可以通过指定 salt 来控制最终合约地址的尾号（如 888、666）。 它只需要部署一次，之后所有用户都能通过它来部署带定制尾号的代币。

📋 部署后的配置步骤

  1. 1
    部署 VanityFactory 合约
    点击上方"部署 VanityFactory"按钮，在 MetaMask 中确认交易。
  2. 2
    复制合约地址
    部署成功后，复制显示的 Factory 合约地址。
  3. 3
    更新 common.js 配置
    在 frontend/common.js 中，将对应链 ID 的地址替换为刚部署的地址：
    '97': '0x0000000000000000000000000000000000000000', // BSC 测试网 '56': '0x0000000000000000000000000000000000000000', // BSC 主网
  4. 4
    刷新部署页面
    打开或刷新 deploy.html，尾号定制功能即可使用。
